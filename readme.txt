# Inventory Management App

A Flask-based web application to manage inventory of products across multiple warehouses.

## Features

- Add/Edit/View Products
- Add/Edit/View Locations
- Track Product Movements
- View Inventory Balance Report

## How to Run

1. Clone the repo: `git clone <repo-url>`
2. Create a virtual environment: `python -m venv venv`
3. Activate venv: `venv\Scripts\activate`
4. Install dependencies: `pip install flask flask_sqlalchemy`
5. Run: `python app.py`
6. Open browser: `http://127.0.0.1:5000/`
